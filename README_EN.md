# WikiPopularity

Web application that compares the popularity of Wikipedia articles based on the number of annual page views, using real-time data from the Wikimedia API and interactive chart visualization.

This project was developed under the guidance of the instructor [Lincoln Souza](https://github.com/souzalb) for educational purposes, focusing on public API consumption, asynchronous data handling, and dynamic frontend visualization.

## 📌 Overview

WikiPopularity allows users to search for a term and view related Wikipedia articles, displaying a chart with daily page view counts throughout the year.

The project demonstrates:

- Public API consumption
- Asynchronous data handling with async/await
- Data processing in JavaScript
- Dynamic chart visualization
- Responsive UI with Tailwind CSS

---

## ⚙️ Features

- Wikipedia article search
- Filtering of the most relevant results
- Page view data retrieval per article
- Total views calculation
- Interactive chart visualization
- Clickable chart redirecting to the original article
