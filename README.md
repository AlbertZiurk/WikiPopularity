# WikiPopularity

Aplicação web que compara a popularidade de artigos da Wikipédia com base no número de visualizações anuais, utilizando dados em tempo real da Wikimedia API e visualização interativa com gráficos.

Este projeto foi desenvolvido sob tutorial do docente [Lincoln Souza](https://github.com/souzalb) com fins educacionais, com foco no consumo de APIs públicas, manipulação de dados assíncronos e visualização dinâmica no frontend.

## 📌 Visão Geral

O WikiPopularity permite que o usuário pesquise um termo e visualize os artigos mais relacionados da Wikipédia, exibindo um gráfico com a quantidade de visualizações diárias ao longo do ano.

O objetivo do projeto é demonstrar:

- Consumo de APIs públicas
- Manipulação de dados assíncronos com async/await
- Processamento de dados em JavaScript
- Visualização dinâmica com gráficos
- Interface responsiva com Tailwind CSS

---

## ⚙️ Funcionalidades

- Busca de artigos da Wikipédia
- Filtragem dos resultados mais relevantes
- Consulta de dados de visualização por artigo
- Cálculo de total de views
- Exibição gráfica interativa
- Clique no gráfico redireciona para o artigo original
